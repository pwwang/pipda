    </>

    Check if the pipeable is called with piping.

    Example 
    >>> data >> verb(...)>>> data >>= verb(...)
    Parameters
    • pipeable (str) — The name of the verb, used in warning or exception messaging
    • fallback (str) — What if the AST node fails to retrieve?piping - Suppose this verb is called like data >> verb(...) normal - Suppose this verb is called like verb(data, ...) piping_warning - Suppose piping call, but show a warning normal_warning - Suppose normal call, but show a warning raise - Raise an error
    Returns (bool)

    True if it is a piping verb call, otherwise False
